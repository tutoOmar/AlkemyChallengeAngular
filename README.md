# AlkemyChallengeAngular
This is a Challenge of Almeky, this project has been developed on Angular 13, using Bootstrap, using Reactive forms and communicate components.
The challenge consist in an application for a restaurant named "A la carta", the application consume the api "https://spoonacular.com/food-api", that have several dishes and menus. Additional we should to login first, and after you can search dishes, and add to "cart". In your menu section select be the total price, average Health score and averga cooking time.
